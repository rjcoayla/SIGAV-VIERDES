Exportación de datos

Exporta los datos del sistema para análisis externo, respaldo o generación de dashboards.

Power BI — Instrucciones de conexión
  1. Haz clic en "Generar Dashboard Power BI" — descarga un archivo .pbix de plantilla y un .json con los datos.
  2. Abre Power BI Desktop e importa el archivo .json vía Obtener datos → JSON.
  3. Los datos incluyen: OT (número, estado, avance, zona, m²), ejecuciones, operarios y avance por zona.
  4. Puedes programar la exportación periódica desde esta pantalla para mantener el dashboard actualizado.
